# TaskWise

> Smart Web-Based Student Task Management System with Priority and Deadline Analytics

---

## Project Overview

This system helps students organize their academic tasks in one platform. It would allow users to tract assignments, deadlines, and progress. Many students struggle with managing multiple tasks, which leads to missed deadlines. This web app provides a simple solution by centralizing all task-related information.

##  Key Features

- **Task Management (CRUD)** — Create, update, delete, and organize academic tasks..
- **Smart Priority System** — Automatically assigns priority levels based on deadline proximity and task difficulty.
- **Deadline Analytics** — Tracks upcoming, overdue, and completed tasks.
- **Dashboard Overview** — Displays task summaries, progress, and key insights.
- **Progress Tracking** — Shows completion rate and task history.
- **Reminder Notifications** — Alerts users about upcoming deadlines.
- **Task Categorization** — Organize tasks by subject or category.
- **Group Collaboration** — Shared tasks for group projects.
